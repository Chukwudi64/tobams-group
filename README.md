# Tobams Group Training and Development Page

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/Chukwudi64/tobams-group
cd tobams-group
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open your browser and visit:
http://localhost:3000
