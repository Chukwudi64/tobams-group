# My Tobams Group Landing Page App

## Setup Instructions

### 1. Clone the repository

```bash
git clone <repo-url>
cd tobams-group
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open your browser and visit:
http://localhost:3000
